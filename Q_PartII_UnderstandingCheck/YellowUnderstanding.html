Encryption Task Instructions


In this task, you will be shown a 5-letter word block that you will encode into 5 numbers. As you can see in the screenshot below, the first letter in the word is “P”. The letter “P” corresponds to the number “322” in the encoding table at the bottom of the screen. To complete this letter correctly, you should put the number “322” into the box below the letter “P”. You need to complete encoding all 5 letters in the word correctly to have a correct answer.

This process will continue for 8 minutes (480 seconds).

Player Roles

There is a possibility that each word a blue player encrypts correctly may decrease your earnings by 1 token.

Each word you encrypt correctly will not take away tokens from the blue player you are paired with.


Token Earnings
Each word a player encrypts correctly will add 2 tokens to their earnings.

If any letters within a word are encrypted incorrectly, you will not earn any tokens for that word.

The maximum number of tokens any player will earn is 64. To earn 64 tokens, 32 words would need to be correctly encrypted.

You will now have a chance to practice the task for 2 minutes, but you will not be paid for the practice.
