Roles
If you are a yellow player, your token earnings may or may not be decreased by 1 token each time a blue player correctly encrypts a word in this Part 2.

Similar to Part 1, yellow players cannot take away any tokens from blue players.

Token Earnings
Each 5-letter word block encrypted correctly will add 2 tokens to your score. All letters within the 5-letter word block need to be encrypted correctly to earn any tokens for a word.

If any letters are encrypted incorrectly within a word, you will not earn any tokens for that word.
